# Linux服务管理

服务简介与分类

  • 查看所有进程(已运行服务)

RPM包安装服务的管理

独立服务的管理

  • systemctl docker start

  • chkconfig --list | grep httpd
  • chkconfig httpd on (默认级别就是2345)

基于xinetd服务的管理 (超级守护进程)

  • netstat -tlun

源码包安装服务的管理

  • 安装说明在软件目录下的install文件里面

  • chkconfig -add apache (将apache加入chkconfig管理)
